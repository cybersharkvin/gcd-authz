% SUGGESTED TITLE: Constitutive Authorization at the Decoding Boundary: Grammar-Constrained Decoding as a Positive, Generation-Time Security Control for LLM Agents
\begin{abstract}
\label{sec:frontmatter}
We port Galloway's (2020) case for application whitelisting -- a positive, structural control that admits only what is explicitly authorized -- from operating-system execution to large-language-model agent tool calls \cite{galloway2020}. Against a single deployed CTF agent (Tantalus) we replay an outcome-selected corpus of capable authorization-class attacks across nine conditions: a security-neutral Control baseline, four negative/behavioral controls (a defensive prompt \texttt{a1}, an input classifier \texttt{a2}, an output filter \texttt{a3}, and the combined stack \texttt{a4}), a corrective post-parse allowlist gate (\texttt{d}), and the positive/structural control --- grammar-constrained decoding (GCD), in which a per-request grammar \(G_s\) makes out-of-scope tokens ungenerable at the sampler (blind \texttt{c}, with the closed-response \texttt{c\_plus} and least-privilege \texttt{c\_guided} variants). Like Galloway, we score by effect at scale with Clopper--Pearson confidence intervals \cite{clopper1934}; unlike Galloway, we add two layers: a soundness core that quantifies over all logit distributions -- turning an observed \(0\) into a provable \(0\), \(\mathrm{emitted}\in L(G_s)\subseteq\mathrm{Authorized}(s)\) -- and a cost/economics axis. Our thesis is constitutive vs.\ corrective: \texttt{c} and \texttt{d} enforce the same policy over \(L(G_s)\) but differ in enforcement completeness, the empirical signature being emission (the count of times the unauthorized artifact was produced: \(0\) under \texttt{c} by construction, \(>0\) under \texttt{d}). Across three datasets (\(\sim\)2,412,613 trials): on a 1.7B generator\,\(\times\)\,gate factorial the GCD arms show \(0\) emission and an inert gate while behavioral-behind-gate cells produce-then-catch 104k--120k out-of-scope calls each -- the GCD arms running at \(\sim\)40--55\% of those arms' tokens; on the capable non-Qwen Mistral-Small-4-119B anchor \texttt{a4} still leaks (bypass 1.82\% [1.43, 2.29], CI-lower \(>0\)) while \texttt{c} is 0.00\% [0, 0.09], below every behavioral point estimate, with a validity dividend (99.8\% valid vs.\ \(\sim\)59\% native); and an abliterated steelman moves 97.85\% \(\to\) 0\%. Honesty envelope: the headline leak and the formal contrast are carried by a single capable anchor; external validity covers 2 of 3 pretraining lineages (Qwen and Mistral); the registered 2-percentage-point non-inferiority TOST is not yet powered; grammar-compile time is not yet captured; and \texttt{c\_sealed}'s 0\% is partly true by construction.
\end{abstract}
